# Easy to Understand Flutter App

This is a very simple Flutter counter application with easy-to-understand comments in the code. It helps new developers quickly learn how a basic Flutter app works.

## Features
- A simple counter that increments when you tap a button.
- Clean, thoroughly commented code for beginners.

## How to Run
1. Install [Flutter](https://flutter.dev/).
2. Run `flutter pub get` in this directory.
3. Run `flutter run`.
